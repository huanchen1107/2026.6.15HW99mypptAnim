  這份 PPTX 是由 NotebookLM 生成的，實際檢查後發現每張 slide

  因此請你不要假設 PPTX 裡面有 title、text box、shape 或 chart element。

  目標：                                                                        
  1. 將 PPTX 每一頁轉成高解析 PNG。                                             
  2. 對每張 slide image 做 segmentation / layout detection。                    
  3. 將適合動畫化的視覺元素切成透明 PNG layers。                                
  4. 保留每個 layer 的原始座標與尺寸。                                          
  5. 使用 HyperFrames 建立 HTML / CSS / JS video composition。                  
  6. 將每個 layer 疊回原始位置，並依照 storyboard 加入動畫。                    
  7. 為每一頁產生繁體中文女聲旁白。                                             
  8. 將旁白、字幕、動畫時間軸同步。                                             
  9. 最後輸出包含動畫、旁白與字幕的 MP4 影片。

  請優先切出以下類型：                                                          
  - title                                                                       
  - subtitle                                                                    
  - key point card                                                              
  - chart                                                                       
  - icon                                                                        
  - arrow                                                                       
  - illustration                                                                
  - decorative shape                                                            
  - main visual object                                                          
                                                                                
  請避免：                                                                      
  - 把文字拆成單字或筆畫                                                        
  - 把背景紋理切成 layer                                                        
  - 把過小雜訊當成物件                                                          
  - 讓透明 PNG 邊緣出現白邊或黑邊                                               
  - 讓 layer 疊回原圖時產生明顯偏移                                             
                                                                                
  Voiceover / Narration 規則：                                                  
  這支影片需要加入中文女聲旁白。

  - 避免過度機械感                                                              
  - 避免太誇張的廣告語氣                                                        
  - 每頁旁白長度需符合該頁動畫時間                                              
                                                                                
  請為每一頁輸出：                                                              
  1. narration_script.md：每頁旁白文字

  - 當旁白提到標題時，標題 layer 出現

  - 當旁白提到圖表或數據時，chart layer 出現                                    
  - 當旁白提到關鍵結論時，highlight 或 conclusion layer 出現                    
  - 每一頁旁白結束後，保留 0.3 到 0.8 秒停頓再進入下一頁                        
                                                                                
  音訊要求：                                                                    
  - Voice: female                                                               
  - Language: zh-TW / Traditional Chinese                                       
  - Format: MP3 或 WAV                                                          
  - Sample rate: 44.1kHz 或 48kHz                                               
  - 音量需標準化，避免忽大忽小                                                  
  - 背景音樂若有，音量需低於旁白，不可蓋過人聲

  技術流程：                                                                    
  1. Convert PPTX to PNG, 1920x1080 or higher.                                  
  2. Use layout detection to identify large visual regions.                     
  3. Use SAM / Grounded SAM / segmentation refinement to create masks.          
  4. Export each object as transparent PNG.                                     
  5. Generate metadata.json for each slide.                                     
  6. Generate narration_script.md for all slides.                               
  7. Generate narration_timing.json to sync voiceover and layer animations.     
  8. Generate TTS audio files using a natural female zh-TW voice.               
  9. Generate subtitles.srt or subtitles.vtt.                                   
  10. Generate HyperFrames composition using HTML, CSS and JavaScript.          
  11. Use GSAP or CSS animation for motion.                                     
  12. Sync layer animation with narration timing.                               
  13. Preview before rendering.                                                 
  14. Render final MP4 with animation, voiceover and optional subtitles.
