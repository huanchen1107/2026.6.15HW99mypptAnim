## Important V4 Change

Before:

The summary table emphasized global best subsets by feature count, which made it look like algorithms reused the same feature set.

After:

The summary now lists each algorithm's own top-5 selected feature set, with RMSE and R-squared.

New focus:

Compare per-algorithm selected features instead of only global best feature combinations.

## Key Results

- Strongest feature: `R&D Spend`
- `R&D Spend` ranked top 1 by `10/10` algorithms
- `Marketing Spend` is usually selected as an important supporting feature
- `Administration` is often selected, but should still be treated cautiously as possible operating-cost noise
- State dummy variables should not be over-interpreted because the dataset has only 50 rows
